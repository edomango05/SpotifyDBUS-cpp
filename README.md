# Very basic c++ project template
Includes `.vscode` project configuration for **Intellisense**, project structure and a customizable `Makefile`.
## Makefile commands
The following commands can be used with this Makefile:

`make all`

`make clean`

`make program`

`make build`

`make release`

`make debug`

`make info`
